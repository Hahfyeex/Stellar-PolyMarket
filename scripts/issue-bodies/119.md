## Context
Seeing other users actively betting builds trust and FOMO, two key drivers of prediction market engagement.

## Implementation Guide
1. Create a SocialTicker component rendering a horizontal scrolling strip
2. Connect to GET /api/activity/recent endpoint (polling every 10s) or WebSocket stream
3. Each ticker item: User [ABCD...XYZ] staked [amount] XLM on [outcome] — [market title]
4. Truncate wallet: addr.slice(0,4) + '...' + addr.slice(-3)
5. Use Framer Motion AnimatePresence with motion.div for entry/exit animations
6. Add onMouseEnter to pause animation and onMouseLeave to resume

## Guidelines
- Scrolling ticker showing recent bets with anonymized wallet addresses
- Framer Motion for smooth entry/exit animations
- Feed via WebSocket or 10-second polling
- Ticker pauses on hover
- Key requirement: Framer Motion / Real-time Feed
