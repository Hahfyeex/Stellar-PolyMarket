## Context
Users want to understand potential returns before committing funds. A simulator removes uncertainty and encourages larger, more confident stakes.

## Implementation Guide
1. Add a collapsible WhatIfSimulator panel below the bet form on market detail page
2. Input: a range slider and a number input field, kept in sync
3. Formula: projectedPayout = (stakeAmount / (poolForOutcome + stakeAmount)) * totalPool * 0.97
4. Render a BarChart from Recharts with two bars: Projected Win and Stake Amount
5. Show implied probability: (poolForOutcome / totalPool * 100).toFixed(1)%
6. Update all values on every slider/input change using useEffect with 200ms debounce

## Guidelines
- Collapsible simulator panel on market detail page
- Stake input via slider and text field
- Projected payout and loss displayed as Recharts bar chart
- Real-time update on input change
- Show implied probability alongside projected return
