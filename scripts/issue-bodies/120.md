## Context
New users unfamiliar with prediction markets or Stellar wallets need guided onboarding. Drop-off at wallet connection is high without it.

## Implementation Guide
1. Create an OnboardingWizard component with a currentStep state (0-3)
2. Step 1: Render wallet connect button using useWallet hook
3. Step 2: Static explainer card with market mechanics diagram
4. Step 3: Render a demo MarketCard with a disabled bet form
5. Step 4: Payout explainer with a simple example calculation
6. Persist onboardingComplete to localStorage on final step or skip
7. Check localStorage on app mount and skip wizard if already complete

## Guidelines
- 4-step wizard covering wallet, markets, betting, and payouts
- Progress stepper at top of each step
- Skip option that jumps to dashboard
- Persist completion in localStorage
- Key requirement: 4-step Progress Stepper / Logic

## Definition of Done
- [ ] All 4 steps render correctly and navigate forward/backward
- [ ] Skip button jumps directly to dashboard from any step
- [ ] Completion state persists in localStorage and wizard does not re-show on refresh

- [ ] Each step has a distinct illustration or icon
- [ ] Step navigation logic is unit tested at more than 90% coverage
- [ ] localStorage key structure is documented with inline comments
- [ ] UI matches color palette and spacing system
