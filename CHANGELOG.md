## [0.1.0] - 2026-03-21

### Added
- Initial project scaffold (frontend, backend, contracts, oracle)
- Next.js frontend with Tailwind CSS and Freighter wallet integration
- Node.js / Express API with PostgreSQL support
- Soroban smart contract: create market, place bet, resolve, distribute rewards
- Oracle service with CoinGecko integration for crypto price markets
- PostgreSQL schema for markets and bets
- README with full project documentation
- CONTRIBUTING, CODE_OF_CONDUCT, LICENSE
- GitHub issue and PR templates
